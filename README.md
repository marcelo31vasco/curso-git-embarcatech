# Disciplina de GIT - EMBARCATECH

# Repositório de Atividades do Curso de Git

Este repositório foi criado para documentar e compartilhar as atividades realizadas no curso de Git oferecido pelo **IFRN**, como parte do programa de **Residência em Sistemas Embarcados e Internet das Coisas – Embarcatech**.

## Sobre o Curso

O curso é ministrado pelo professor **Mizael Cortez** e abrange os fundamentos do Git, uma ferramenta essencial para controle de versão e colaboração em projetos de software.

## Acesso ao Conteúdo

Você pode assistir às aulas e acompanhar o conteúdo do curso através do seguinte link no YouTube: [Aulas de Git - Professor Mizael Cortez](https://www.youtube.com/watch?v=uimSsA6362I)

## Contribuições

Sinta-se à vontade para explorar, contribuir e aplicar os conceitos aprendidos. Este repositório é um espaço aberto para aprendizado e colaboração!
